**基于深度学习与特征工程的端到端无人驾驶技术架构研究**

**引言**

在自动驾驶领域的快速发展下，端到端的深度学习方法表现出巨大的潜力，成为实现自动驾驶系统的关键路径之一。这类别的方法直接将感知输入，如摄像机图像，映射到控制命令，如转向和加速，从而大幅减少了对复杂特征工程的依赖。然而，这种方法面临着一些限制：目前通过模仿学习训练出的驾驶策略在测试时无法控制，这意味着无法在即将到来的交叉路口引导车辆进行特定的转弯，限制了这些系统的实际应用价值。鉴于此，本研究提出一种基于条件模仿学习的方法，并在高清命令输入的基础上进行扩展。在测试时，所训练的驾驶策略充当一个响应动作协调的司机，同时仍然能够响应导航命令。

模仿学习作为一种具有前景的训练自动驾驶系统的方法，得到了越来越多的关注。通过人类驾驶的示范来绘制易于大量收集的地图，模仿学习可以用来训练一个模型，该模型将感知输入映射到控制命令。这种方法已应用于车道跟踪和越野避障等场景。尽管如此，这些系统存在显著的局限性，例如Bojarski等人训练的网络仅在日常车道和道路跟踪中有效。当需要变道或从一个道路转向另一个道路时，需要人类驾驶员控制，表明模仿学习在处理复杂交互场景中的局限性。

尽管如此，模仿学习已被应用到各种任务中，如拟人运动控制、自主飞行、模拟导航行为、越野驾驶和路跟踪等。这些应用在输入形式（原始感知输入或手工制作的特有特征）、预测控制信号、学习算法和学习表示方面存在技术差异。相关研究如Pomerleau、LeCun及其同事所进行的工作，采用地面车辆，训练深度网络从摄像机输入预测驾驶员的动作，并集中于如车道跟踪或障碍物避让这样的纯反应性任务。与此相较，本研究开发了一种命令条件化表述，使模仿学习能够应用于更复杂的城市驾驶场景。另一个差异在于，我们不仅要学习控制方向，还要学习控制加速度。

紧邻层次化方法的是学习多用途和参数化控制器的想法。在机器人领域，参数化目标已被用于训练运动控制器。Schauel等人提出了一个用于具有参数化值函数的强化学习的一般框架，这些函数在状态和目标之间共享。研究也探讨了在三维环境中导航的参数化目标家族。这些研究表明，通过条件化的方式来训练控制器，是一个有前景的方向。然而，这些研究与本工作在模型架构、应用领域和学习方法上有显著不同，本工作聚焦于基于视觉的自动驾驶，并采用条件模仿学习。

为了使无人驾驶系统能够适应更复杂多变的场景，并在测试时能够响应驾驶指令，本研究提出了将深度学习和特征工程结合起来的端到端无人驾驶技术架构。具体来说，我们评估不同架构在条件模仿学习中的视觉驾驶方式，并通过在三维模拟城市驾驶环境和1/5比例的电动卡车上进行实验，验证这些方法的有效性。两种系统都基于视觉输入行驶，但同时仍能对高级导航命令做出响应。本文阐明了模型架构、学习异常处理、性能优化等方面的贡献，旨在为端到端的自动驾驶系统设计提供新的视角和解决方案，推动自动驾驶技术向更高水平的发展。
